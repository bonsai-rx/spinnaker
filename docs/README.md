# Bonsai - Spinnaker Library

A Bonsai library for interfacing with FLIR cameras using the Spinnaker SDK version 4.2.0.83.

## How to Use

To use `Bonsai.Spinnaker` for visual reactive programming, please install this package using the [Bonsai package manager](https://bonsai-rx.org/docs/articles/packages.html).

## Additional Documentation

For additional documentation and examples, refer to the [official Bonsai documentation](https://bonsai-rx.org/spinnaker).
